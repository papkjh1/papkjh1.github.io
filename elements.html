#BOM Project

'BOM' 프로젝트란 ?

1인 미디어 형의 마이크로 블로그 사이트이다.

인터넷을 통해 실시간 커뮤니케이션을 할 수 있는 프로그램 . 일종의 채팅 프로그램을 가지고, 현재 무엇이 일어났는지 , 무엇을 생각하지는지를 매우 간단하게 기술하고 남들이 이를 구독한다.

##개발환경 : STS(eclipse) , SQL Developer

##인원 : 8명

##사용언어 : JAVA, CSS, HTML5, JAVAScript, JQuery, Ajax, MyBatis, Spring

##본인 역할 : 통합검색으로 모든 글의 검색게시판 구현 '트위터'탐색하기 오마주

1.통합검색 기능 구현

-사용자,인기,최신글,사진,동영상 네비클릭시 해당 게시글 출력

-차단회원 게시글 차단출력

2.검색기록 구현

-사용자의 검색기록 남기기 및 전체삭제 클릭시 자동검색 구현

TimeLine 에서 검색View로 접근하는 모습 , 검색어 입력과 함께 해당 데이터를 출력하는 모습.

searhcView 에서 검색어 입력후 인기,사용자,최신,사진,동영상 출력하는 모습.

searhcView 에서 검색기록을 남긴 모습과 최신화 , 그리고 전체삭제까지 실행하는 모습.
